# Solfege

Solfege is a Roguelike game that focus on an Aspiring composer gets stuck in the middle of a zombie apocalypse,
he must use his wand to compose his instruments to fight for him, use sheet music to conduct powerful spells 
while making sure he is in time, and much more if he wants to make it out alive
